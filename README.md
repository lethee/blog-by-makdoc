# Blog

## Clone

    git clone git@github.com:lethee/blog-by-makdoc.git
    cd blog-by-makdoc
    npm install -g gulp
    npm install -g bower
    npm install
    bower install
    gulp --tasks
    gulp server

## Init

    npm init
    npm install --save git://github.com/pismute/gulp-makdoc.git
    vi .gitignore
    vi .jshintrc

## Init contents

    vi app/layouts/default.html.hbs
    vi app/root/index.html.hbs
    vi app/root/index.html.hbs.json
